CSS

Copie y pegue la hoja de estilo <link> antes que <head> todas las demás hojas de estilo para cargar nuestro CSS.

<link href="http://127.0.0.1:8000/static/chatbot/css/style.css" rel="stylesheet" type="text/css">
<link href="http://127.0.0.1:8000/static/chatbot/css/chatBot.css" rel="stylesheet" type="text/css">

JS

Muchos de nuestros componentes requieren el uso de JavaScript para funcionar . Coloque una de las siguientes <script> cerca del final de sus páginas, justo antes de la etiqueta de cierre </body> para habilitarlas.

<script src="http://127.0.0.1:8000/static/chatbot/js/chatbot.js" crossorigin="anonymous"></script>
<script src="http://127.0.0.1:8000/chatbot/js/chatbot_integration.js" crossorigin="anonymous"></script>
